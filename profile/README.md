## 二、新手必读：搬瓦工VPS的几个核心概念

在真正了解hk85之前，先简单过一下几个新手必须懂的概念，不然看到那些专业术语脑子里会转圈圈。

**CN2 GIA是什么？**
中国电信的顶级骨干网线路，延迟低、丢包少，是目前国内到海外速度最稳定的线路之一。缺点是贵。

**CMI是什么？**
China Mobile International，也就是中国移动国际线路。对移动用户来说，CMI直连的延迟表现相当优秀，甚至在部分场景下不输CN2 GIA。

**限量版套餐是什么意思？**
搬瓦工的限量版套餐，顾名思义就是数量有限、低价高配，属于"买到就是赚到"那种。同样配置的常规套餐往往是限量版的2-3倍价格，但限量版常年缺货，补货之后通常几小时内就被抢光了。

**KiwiVM是什么？**
搬瓦工自研的VPS管理后台，可以重装系统、查看流量、一键迁移机房等。新手也能很快上手。
